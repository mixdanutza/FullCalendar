# FullCalendar
Open source JavaScript jQuery plugin for a full-sized calendar built using: 

<ul>
    <li> FullCalendar</li>
    <li> Moment.js </li>
    <li> Java Script </li>
    <li> Angular JS </li>
    <li> HTML/CSS </li>
    <li> jQuery </li>    
</ul>
